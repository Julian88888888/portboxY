# Як редагувати профіль

## Де знаходиться форма редагування?

1. Увійдіть в систему (залогіньтесь)
2. Перейдіть в **Dashboard** (Панель управління)
3. Виберіть вкладку **"Profile"** (Профіль) - це перша вкладка

## Як редагувати профіль?

### Якщо профіль ще не створений:

1. Ви побачите повідомлення: **"Створіть свій профіль"**
2. Заповніть форму:
   - **Username** (Ім'я користувача) - обов'язкове, 3-30 символів, тільки літери, цифри, крапки та підкреслення
   - **Name** (Ім'я) - ваше повне ім'я або назва бізнесу
   - **Job Type** (Тип роботи) - виберіть один з типів: Model, Photographer, Wardrobe Stylist, тощо
   - **Profile Description** (Опис профілю) - опишіть себе
3. Натисніть **"Створити профіль"**

### Якщо профіль вже існує:

1. Всі поля будуть заповнені вашими даними
2. Змініть потрібні поля
3. Натисніть **"Зберегти зміни"**

## Завантаження фото

### Фото профілю (Avatar):
1. Натисніть на зображення під **"Add Profile Photo/Logo"**
2. Виберіть файл (максимум 5MB)
3. Фото завантажиться автоматично
4. Щоб видалити фото, натисніть кнопку **"Remove"**

### Фото заголовка (Header):
1. Натисніть на зображення під **"Add Profile Header Photo"**
2. Виберіть файл (максимум 5MB)
3. Фото завантажиться автоматично
4. Щоб видалити фото, натисніть кнопку **"Remove"**

## Валідація

### Username (Ім'я користувача):
- ✅ Мінімум 3 символи, максимум 30
- ✅ Тільки літери (a-z, A-Z), цифри (0-9), крапки (.) та підкреслення (_)
- ✅ Без пробілів
- ✅ Має бути унікальним (не використовується іншим користувачем)

### Помилки:
- Якщо username вже зайнятий, ви побачите помилку: **"Username is already taken"**
- Якщо формат неправильний, ви побачите помилку з описом правил

## Статуси збереження

- **"Збереження..."** (синій) - дані зберігаються
- **"Збережено ✓"** (зелений) - дані успішно збережені
- **"Помилка збереження"** (червоний) - сталася помилка

## Важливо

✅ **Зміни зберігаються миттєво** - після збереження дані одразу відображаються на Dashboard без перезавантаження сторінки

✅ **Профіль створюється автоматично** - якщо профілю немає, він створиться при першому збереженні

✅ **Фото зберігаються в Supabase Storage** - всі фото безпечно зберігаються в хмарі

## Якщо бачите "Loading profile..."

Це означає, що:
1. Профіль завантажується з бази даних (перший раз)
2. Або профіль ще не створений

**Рішення:**
- Зачекайте кілька секунд
- Якщо "Loading..." не зникає, перезавантажте сторінку
- Якщо профіль не існує, форма все одно з'явиться для створення профілю

## Проблеми?

### Профіль не зберігається:
1. Перевірте, чи всі обов'язкові поля заповнені
2. Перевірте валідацію username
3. Перевірте консоль браузера на наявність помилок

### Фото не завантажується:
1. Перевірте розмір файлу (максимум 5MB)
2. Перевірте формат (тільки зображення: JPG, PNG, тощо)
3. Перевірте, чи налаштовані storage buckets в Supabase (див. STORAGE_SETUP.md)
